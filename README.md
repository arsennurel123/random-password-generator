# 🔐 Advanced Password Generator CLI

![Python Version](https://shields.io)
![License](https://shields.io)

Профессиональное консольное приложение на Python для генерации криптостойких (учебных) паролей. Проект выполнен в рамках учебного курса по информатике.

## 🌟 Особенности
- **Кастомизация**: Генерация пароля любой длины.
- **Безопасность**: Использование полной таблицы ASCII (буквы, цифры, символы).
- **Чистый код**: Соблюдение стандартов PEP 8.
- **Open Source**: Код открыт для аудита и улучшений.
